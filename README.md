# Era dos Fidalgos

Um jogo de estratégia (WIP) no qual o jogador deve gerenciar seu reino e obter
aliados para aumentar seu poderio.

## Funcionalidades a Testar

- Se os menus funcionam como esperado;
- Os eventos funcionam e são interativos;
- Se o mapa pode ser navegado e o jogador pode explorar os dados;
- Se o jogador perde quando um dos três estados ganha ou perde muito poder (-100 ou +100).
